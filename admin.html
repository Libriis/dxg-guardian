GitHub token na publikovanie

Aby si vedel zmeny publikovať priamo na web, treba jednorazovo vložiť GitHub fine-grained token. Uloží sa len v tomto prehliadači (localStorage), nikam inam sa neposiela.

  1. Otvor github.com/settings/personal-access-tokens/new
  2. Repository access → Only select repositories → Libriis/dxg-guardian
  3. Permissions → Repository permissions → Contents → Read and write
